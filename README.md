CLIPMAKER by Gemini
Imagine que você tem um vídeo super comprido da sua festa de aniversário, mas só quer aquela partezinha engraçada onde o bolo cai para postar no TikTok. Em vez de você ficar procurando e cortando manualmente, o ClipMaker faz isso para você usando uma IA (Gemini).

Como funciona?
O intuito da ferramenta é o recorte de vídeos com o objetivo de serem virais. O Cloudinary é a ferramenta responsável por armazenar e transcrever o audi/vídeo para que o Gemini possa ler e recortar o momento de maior impacto e viralização. Com a utilização do framework Tailwind e GSAP o site ganha um pouco mais de vida.
Para que o programa rode é necessário configurar a cloudname e presets feitos diretamente no cloudnary e inserir uma API Key do gemini válida.
